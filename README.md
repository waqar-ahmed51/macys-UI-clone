# macys-UI-clone
For practice created the clone of 'https://www.macys.com/shop/search?keyword=jeans'.
I created this during my internship program at NISUM Karachi. This repository is made for record purposes.
I used HTML, CSS, Javascript, and Bootstrap for this clone.

Complete Macy's website clone which includes:
1. Topmost bar which includes links for different functionalities.
2. Main logo bar includes search option UI and chart icon.
3. Navbar with 8 customized CSS dropdowns which open on hover and show a full-size mega menu.
4. Megamenu with category heading and category items in it.
5. Side Navbar for product filtering purposes.
6. Side Navbar with collapsable buttons with custom CSS for a minimal look.
7. Checkboxes for product filtering in each side navbar button.
8. Main product showcase layout for product showing.
9. Product display with customized bootstrap cards.
10. Product cards start a slide show of the product images on hover.
11. Product card with image radio buttons for changing the product image color property.
12. Product card includes start rating with the total number of feedback.
13. Product cards include customized details about the product.
14. Good looking footer having different portion for different links divided into different rows and coloumns.
